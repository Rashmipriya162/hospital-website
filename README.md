🏥 Hospital Management System (Flask Project)

This is a web-based hospital management system developed using Python Flask. 
The project allows patients to book appointments online and enables admin to view all patient records.

✨ Features:
- Patient Appointment Booking Form
- Stores patient data (name, age, gender, contact, address, problem)
- Admin Panel to view all patient records
- Clean and responsive UI using Bootstrap

🛠️ Tech Stack:
- Frontend: HTML, CSS, Bootstrap
- Backend: Python (Flask)
- Storage: Text File (data.txt)

🌐 Future Improvements:
- Add Admin Login System
- Use Database (SQLite/MySQL)
- Deploy with cloud hosting
